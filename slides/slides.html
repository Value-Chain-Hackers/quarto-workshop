Key features

  • Language agnostic (Works with R, Python, Julia, Observable):
    • Can combine different languages in a single document
    • Same syntax
    • One command line
  • IDE Agnostic (Integrations - see getting started)
    • RStudio
    • Visual Studio Code/VS Codium
    • Jupyter
  • Diverse outputs
    • Formats: pdf, html, epub, docxâ¦
    • Media: articles, presentations (like this one!), blogs, websites, booksâ¦
  • Sensible defaults
    • Bootstrap
